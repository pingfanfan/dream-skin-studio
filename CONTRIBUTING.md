# Contributing

Code contributions must be original or compatible with the MIT license.

Public skin contributions must include:

- a `rights.json` file;
- a clear rights basis and redistribution license;
- SHA-256 fingerprints for every visual asset;
- no third-party character, real-person likeness, Logo, watermark, copied UI,
  or living-artist/studio imitation.

Run `npm test` before opening a pull request. Packs with unclear rights or
failing fingerprints are rejected automatically.
