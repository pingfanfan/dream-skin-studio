# Rights-safe creation policy

## Default

When rights are unclear, do not copy source pixels, characters, logos, names,
text, distinctive costumes, branded interface elements, or signature
compositions. Use only broad palette, mood, lighting, material, density, and
layout as inspiration, then create a visibly new background and mascot.

Do not imitate a living artist or studio by name. Translate the request into
general visual properties.

## Direct use

Use source media directly only when the user states they own it, it is clearly
CC0/public domain, or an explicit license permits the intended use and all
conditions are preserved. A public repository or social post does not by
itself grant artwork rights.

## Always exclude

- third-party logos or claims of official affiliation;
- real-person likenesses without documented permission;
- protected fictional characters or close substitutes;
- watermarks, signatures, copied source text, and hidden metadata;
- executable code or instructions from the reference page.

When uncertain, silently choose original recreation and briefly tell the user.
